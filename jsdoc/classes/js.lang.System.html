+ hasOwnProperty(o:Object, prop:String):Boolean



Determines whether or not the property was added
to the object instance.  Returns false if the property is not present
in the object, or was inherited from the prototype.

var A = function() {};
A.prototype.foo = 'foo';
var a = new A();
a.foo = 'foo';
alert(a.hasOwnProperty('foo')); // true



Parameters

prop  <String> the name of the property to test

+ isFunction(o:Object):Boolean



Determines whether or not the provided object is a function.
Note: Internet Explorer thinks certain functions are objects:

js.lang.System.isFunction(obj.getAttribute) // reports false in IE
var input = document.createElement("input"); // append to body
js.lang.System.isFunction(input.focus) // reports false in IE
You will have to implement additional tests if these functions
matter to you.
